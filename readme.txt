智能日程生成器：

简介:
这是一个智能日程生成器，能根据是所在时间和天气状况帮你安排一天的日程(接入Deepseek，使用了flask框架开发后端)

简介:
这个项目是我在学习相关技能时完成的练习案例，发出来记录一下我的学习过程QWQ


项目演示及效果展示:
https://www.bilibili.com/video/BV11PeEzwEeK/




使用：
要求安装python 3.10版本(附带安装包）

使用前先右键Time_help.py选择记事本编辑，把 api = "sk-xxxx" 里面的key换成你的deepseek api key，也可以使用原本提供的key(总限额10yuan用完就没了)

可前往 deepseek开发平台获取key: https://platform.deepseek.com/api_keys

想替换网页背景图片，直接去static删除原背景，然后放入图片命名为top.png即可

先运行安装依赖，再点击run运行主程序。

本地访问: 浏览器输入 127.0.0.1:10001
